# AYEN

텍스트 기반 선택형 RPG 웹게임

---

## 📝 프로젝트 소개

1. **수행 기간** : **2025.05.08 ~ 2025.06.30** (총 약 2개월)
2. 본 프로젝트는 **서울 2033 - Banjiha Games(반지하 게임즈)를 벤치마킹**한 프로젝트 입니다.
3. 플레이어의 선택에 따라 스토리가 다양하게 전개되고, 여러 엔딩을 경험할 수 있도록 설계하고
   **텍스트 중심**의 깊이 있는 이야기와 세계관을 통해 높은 몰입감을 제공하는 것이 프로젝트의 목표입니다.

---

## ✨ 핵심 기능

- **다중 분기 시나리오 엔진**: 사용자의 선택에 따라 데이터베이스 기반으로 실시간 시나리오가 분기되는 로직 구현
- **OAuth 2.0 소셜 로그인**: 카카오 및 구글 API를 연동하여 별도의 회원가입 없이 간편한 서비스 접근 제공
- **업적 및 수집 시스템**: 시나리오별 특정 조건 달성 시 업적을 부여하고, 이를 마이페이지에서 컬렉션 형태로 확인
- **실시간 랭킹**: 사용자가 획득한 업적 개수와 점수를 기반으로 한 전체 사용자 순위 산정 시스템
- **반응형 웹 디자인**: 다양한 디렉션과 디바이스 환경에서도 몰입감을 유지하는 깔끔한 텍스트 중심 UI
